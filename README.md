## macOS_S_CustomTableview
## テーブルビューをプログラムにより作成する

テーブルビューをインタフェースビルダーを使わず、全てプログラムのコードにより作成してみた。これによって、テーブルビューの機能は活用しつつ、主にテーブルビューの外観について、比較的自在に調整・変更が可能になる。

macOS Mojava 10.14.6 / Xcode 11.3.1 / Swift 5.0
